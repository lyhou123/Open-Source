រចនាសម្ព័ន្ធឯកសារ

Zeikki ធ្វើតាមរចនាសម្ព័ន្ធម៉ូឌុល និងរៀបចំ ដែលលើកកម្ពស់ភាពច្បាស់លាស់ និងភាពងាយស្រួលនៃការប្ដូរតាមបំណង។


1.Root directory

នៅក្នុងឯកសារដើម អ្នកនឹងរកឃើញឯកសារសំខាន់ៗ និងការកំណត់ដែលបង្កើតជាមូលដ្ឋានគ្រឹះនៃគេហទំព័ររបសពួកយើង។


Directory សំខាន់ៗគឺជាចំណុចចាប់ផ្តើមសម្រាប់គំរូទាំងមូលរួមមាន:


  • index.html : ឯកសារ index.html បម្រើជាចំណុចចូលសំខាន់សម្រាប់គេហទំព័ររបស់អ្នក។ វាមានមូលដ្ឋានគ្រឹះរចនាសម្ព័ន្ធ និងធាតុដែលកំណត់គេហទំព័រ។

  • styles/ : រចនាប័ទ្ម/ថតត្រូវបានដាក់ដល់កន្លែងផ្ទុកឯកសារ CSS ទាំងអស់។ ឯកសារទាំងនេះកំណត់រចនាប័ទ្មដែលមើលឃើញនៃគំរូ ធានាបាននូវចំណុចប្រទាក់អ្នកប្រើដែលស្អិតរមួត និងទាក់ទាញដែលអាចមើលឃើញនៅលើគេហទំព័រ។

  • scripts/ : scripts/ directory កន្លែងផ្ទុកឯកសារ JavaScript ដែលនាំមកនូវការធ្វើអន្តរកម្មធ្វើអោយគេហទំព័រកាន់តែទាក់ទាញនិងរស់រវើក។ script ទាំងនេះបង្កើនបទពិសោធន៍អ្នកប្រើប្រាស់ និងបន្ថែមមុខងារទៅទិដ្ឋភាពផ្សេងៗនៃគេហទំព័រ។

  • Images/ : រូបភាព/ថតគឺជាឃ្លាំងសម្រាប់ផ្ទុករូបភាពទាំងអស់ដែលបានប្រើនៅទូទាំងគំរូពីនិមិត្តសញ្ញាទៅរូប ភាពផ្ទៃខាងក្រោយ នេះគឺជាកន្លែងដែលទ្រព្យសម្បត្តិដែលមើលឃើញនៃគេហទំព័ររបស់អ្នកត្រូវបានរក្សាទុក។
